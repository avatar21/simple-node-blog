# TypeScript (Type for Javascript)

- Gulp.js (like Maven or Gradle in Java project)
- TSD replaced by Typings: 
    - https://github.com/DefinitelyTyped/tsd/issues/269
    - $ npm uninstall tsd
    - $ rm -rf typings
    - $ npm install typings
    - $ typings init --upgrade
    - $ rm tsd.json
    - $ typings install
